# Frontend User Manual

## Overview
The web UI provides access to the assistant, disease detection, soil monitoring, fertilizer recommendations, and the social feed.

## Pages and features
- Home: product overview and entry points to features
- Login: sign in or create an account
- Dashboard: quick stats and shortcuts
- Disease Detection: upload a cotton leaf image and view predictions
- Soil Monitor: enter NPK values and see soil history
- Assistant: ask questions in Urdu, Punjabi, or English
- Social: create and browse farmer posts

## Typical workflow
1. Open the web app in a browser.
2. Log in to unlock protected features (posting, soil history, saved chats).
3. Use Disease Detection to upload a leaf image and view results.
4. Use Soil Monitor to record NPK values and request fertilizer guidance.
5. Use Assistant to ask agriculture questions and optionally listen to answers.
6. Use Social to create posts, like, and comment.

## Notes
- The assistant uses the RAG service and may respond slowly while the index loads.
- Saved chat history appears only after login.
- Uploads are limited to image files.
